## 🚀 QUICK START (5 Steps)

### Step 1: Setup Backend
```bash
cd backend
npm install
npm run seed
npm run dev
```

### Step 2: Setup Frontend
```bash
cd frontend
npm install
npm run dev
```

### Step 3: Create Manual Directories
```powershell
# Run in backend/uploads/manuals
mkdir PQF01, PQF02, PQH03, ... DPT28
```

### Step 4: Add Your PDF Manuals
```
Place each manual.pdf in:
backend/uploads/manuals/{SERIAL_NUMBER}/manual.pdf
```

### Step 5: Access Application
```
Frontend: http://localhost:3000
Backend: http://localhost:5000/api
```

## 🔧 TECHNICAL SPECIFICATIONS

**Backend Stack:**
- Node.js + Express.js + TypeScript
- PostgreSQL Database
- Socket.io (Real-time WebSocket)
- JWT Authentication
- ExcelJS (Excel Generation)

**Frontend Stack:**
- React + TypeScript + Vite
- Tailwind CSS (Light theme)
- Socket.io Client
- Zustand (State management)
- Framer Motion (Animations)
- Recharts (Charts)
- react-pdf (Manual viewer)

**Database:**
- 5 tables (users, equipment, bookings, calibration, learning_content)
- 28 pre-loaded equipment items
- 4 demo users (operator, technician, supervisor, admin)

**File Storage:**
- Serial number-based organization
- Manual PDFs: `backend/uploads/manuals/{SERIAL}/manual.pdf`
- Product images: `backend/uploads/product-images/{SERIAL}.jpg`
